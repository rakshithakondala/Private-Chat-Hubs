
# Private Chat Hubs

A real-time chat room web application where the **admin creates a private room** and shares a secure **invite code** with users.  
Users can join using the code and chat instantly using Socket.IO for real-time communication.

✅ Admin creates room  
✅ Secure invite codes  
✅ Socket.IO real-time messaging  
✅ No installation required for clients

Tech Stack:
• Node.js
• Express
• Socket.IO
• HTML, CSS, JavaScript
